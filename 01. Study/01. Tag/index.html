html 태그의 6가지


div, p, img, input, button, a 태그
  1. div : 영역을 나누는데 쓰임
  2. p : 문단을 나누는데 쓰임(div 태그와 거의 비슷한 용도이나 단지 밑에 공간이 좀더 생김)
  3. img : 이미지 넣는데 쓰임. 단일 태그임
  4. input : 검색창처럼 사용자에게 입력을 받는 태그 단일태그임
  5. button : 버튼을 만듬
  6. a : 링크를 연결해서 여러페이지로 이동할 수 있음

head 태그 안에 있는 태그들 정리

  1. meta : 웹서버와 웹브라우저 사이에 주고받는 정보를 정의하는데 사용됨 문서 자체의 특성을 나타냄
  2. title : 웹사이트가 열리면 상단에 생기는 tab에 이름을 나타냄
  3. link : 필요한 스타일이나 아이콘등 외부 문서를 html문서에 연결시키는데 사용됨
